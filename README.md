# Logbook iOS App

A SwiftUI-based iOS application for tracking daily work activities and projects.
Designed to work with the Logbook backend API.

## Features
- User authentication
- Create and manage logbook entries
- Create and manage your to do list
- Project-based activity tracking
- Secure token-based API communication

## Tech Stack
- SwiftUI
- MVVM architecture
- URLSession
- iOS 17+

## Requirements
- Xcode 15+
- iOS 17.6+

## Setup
1. Clone the repository
2. Open the project in Xcode
3. Configure API base URL in the app settings
4. Run on simulator or device
